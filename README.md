- 인공지능 영상처리 시스템을 개발하기 위한 Java Progrmming, Python Programming, Database(SQL) 등 프로그래밍 기술 습득
- 영상처리에 대한 이해를 바탕으로 영상처리, OpenCV 등 학습을 통해 AI기반 패션모델 이미지 데이터 전처리 수행
